Intro

shinyBS is an R package that introduces many of the missing components of Twitter Bootstrap into shiny. These include:

  • Tooltips
  • Popovers
  • Alerts
  • TypeAheads
  • Collapse Panels
  • Modal Windows
  • Progress Bars

In addition to adding additional Twitter Bootstrap components, shinyBS also adds table highlighting functions and a menu creation system based on Twitter Bootstrap's navbar component.

Installing shinyBS

shinyBS has been published on CRAN an so can be installed in the usual way:

install.packages("shinyBS")

To get the latest working version you can install from github with devtools:

# To install devtools:
install.packages("devtools")
# To install shinyBS from github:
install_github("ebailey78/shinyBS")

Working with shinyBS

I've tried to make functions in shinyBS work and look, as much as possible, like native shiny functions. Most functions used in ui.R are prepended with bs. So, for example, you can create a popover from within ui.R with:

bsPopover(id = <target_id>, 
          title = "Popover Title", 
          content = "This is the content of the popover.", 
          placement = "below",
          trigger = "hover"
          )
